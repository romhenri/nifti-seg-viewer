# NIfTI Segmentation Viewer

Zero-install browser viewer for NIfTI volumes with segmentation overlays, powered by [niivue](https://github.com/niivue/niivue).

Load a base volume (e.g. an MRI) and overlay a segmentation mask on top of it, with axial / coronal / sagittal / multiplanar / 3D-render views.

**Everything runs locally in your browser, your data never leaves the machine.**
